收获：
1，透明属性rgba和opacity的了解和使用：
rgba是颜色值的透明度，为颜色值设置一定量的透明效果，来达到元素的透明。（子级元素不会继承）
opacity属性，通过设置它的值来达到让元素透明化，元素前提设定好颜色值（子级元素会继承其透明程度）。
2，hover（悬停）属性：元素设定hover属性，
例：
.hiding {
    visibility: hidden（隐藏）;
}
.wrapper:hover>.hiding {
    visibility: visible（显示）;
}
这个例子，鼠标悬停或点击在wrapper元素上是，hiding元素显示，本身hiding元素本身是隐藏的。
